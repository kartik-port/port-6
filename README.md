# Otisium Portfolio

AI/ML Developer Portfolio - Kartik Mhatre

A modern, responsive portfolio website showcasing projects and skills in AI/ML and Full-Stack Development.

## Features

- Modern, responsive design
- Smooth scrolling animations
- Project showcase with detailed information
- Skills and experience sections
- Contact information and social links

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- SCSS
- Lottie Animations

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Start Production Server

```bash
npm start
```

## Deployment

This project is configured for deployment on Netlify.
